# 飞书渠道伙伴资讯 & 培训知识库推送 Skill

> **面向角色：** 字节跳动飞书渠道销售经理
> **核心定位：** 自动整理内容 + 定向推送 + 知识库沉淀
> **技术基础：** 基于 lark-cli（飞书 CLI 工具），集成知识库、即时通讯、文档搜索能力

---

## 一、技能概述

### 1.1 解决的痛点

作为渠道销售经理，日常需要：
- 手动搜集、整理 SaaS 行业新闻发给伙伴
- 同步飞书产品更新到各伙伴群
- 整理培训资料并定向分发给不同能力的伙伴
- 追踪哪些伙伴接收了哪些内容
- 复盘周度推送覆盖情况

**本 Skill 将这些工作全部自动化**：你只需口述/粘贴内容，AI Agent 自动完成分类、归档、生成推送文案、定向分发、记录推送历史。

### 1.2 三大核心模块

| 模块 | 功能 | 对应飞书能力 |
|------|------|------------|
| **模块1：内容归集与分类** | 录入三类内容（行业资讯/产品更新/培训资料），自动打标签并按日期归档 | 飞书知识库（Wiki） |
| **模块2：推送管理** | 生成每日常规推送 / 定向推送 / 紧急推送，支持四个伙伴分组 | 飞书即时通讯（IM） |
| **模块3：知识库查询** | 按标签/日期/关键词检索历史内容，追踪伙伴学习记录，复用优质内容 | 知识库 + 文档搜索 |

### 1.3 三类内容

1. **SaaS 行业每日资讯** — 行业动态、竞品动态、政企数字化政策、市场热点
2. **飞书产品更新** — 新版本功能、AI 能力、定价调整、功能优化、使用技巧
3. **培训资料 / 知识库** — 销售话术、客户方案、行业案例、陪访总结、常见问题 FAQ

## 二、快速开始

### 2.1 前置条件

1. 已安装 `lark-cli`（飞书 CLI 工具）
2. 已完成飞书应用授权（首次使用需执行 `lark-cli config init`）

### 2.2 首次初始化

在 AI Agent 对话中执行：

```
"帮我初始化飞书渠道伙伴内容推送技能"
```

Agent 将自动完成：
1. 在飞书知识库创建「飞书渠道伙伴知识库」空间
2. 创建三大分类目录（行业资讯 / 飞书新功能 / 培训资料）
3. 创建推送记录目录
4. 创建四个伙伴分组群聊
5. 保存配置信息

### 2.3 日常使用

直接对 AI Agent 说：

```
"新增今日SaaS行业资讯：<标题> <内容>"
"新增飞书新功能：<标题> <内容>"
"上传培训资料：<标题> <内容>"
"生成今日推送内容，推送给全体伙伴"
"把培训资料推送给低效伙伴"
"查询近7天所有培训资料"
```

## 四、伙伴分组体系

| 分组 | 定义 | 推送侧重 |
|------|------|---------|
| **全体伙伴** | 所有签约代理商 | 通用信息、产品更新 |
| **核心伙伴** | 主力开单、高产出 | 高级功能、竞争分析、标杆案例 |
| **新签约伙伴** | 合作30天内 | 基础培训、入门资料 |
| **低效伙伴** | 业绩高危/预警 | 销售技巧、商机转化、话术 |

## 六、与已有技能的关系

本 Skill（`lark-channel-sales-content`）与已有 `lark-channel-sales` Skill **互补**：

| 维度 | lark-channel-sales | lark-channel-sales-content |
|------|-------------------|--------------------------|
| 定位 | **数据分析** | **内容管理 + 推送** |
| 核心功能 | 伙伴业绩分级、KPI监测、趋势追踪 | 内容归集、定向推送、知识库查询 |
| 数据方向 | 从伙伴数据 → 发现问题 | 整理内容 → 赋能伙伴 |
| 使用场景 | 周复盘、业绩追踪 | 日常运营、伙伴赋能 |
