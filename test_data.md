# 渠道伙伴常态化信息赋能 Skill — 测试数据集

> **用途：** 验证技能的自动采集、手动录入、分类归档、推送生成、知识库查询全功能。
> **使用方法：** 将以下指令逐条输入到 AI Agent 对话中执行。
>
> ⭐ **v2.0 新增：** 模块0 自动实时采集（WebSearch + WebFetch），确保资讯每日更新、来源可溯。

采集今日全部内容

## 二、测试场景用例

### 场景 1：每日常规推送

**指令序列：**
```
# Step 1: 录入当日三条内容（执行上面三条测试数据）
# Step 2: 生成推送
生成今日推送内容，推送给全体伙伴
```

**验证点：**
- [x] 推送文案聚合了当日所有内容
- [x] 单条内容 ≤ 3 行
- [x] 包含行动引导语
- [x] 发送到了全体伙伴群
- [x] 推送记录已归档

## 三、期望的 Agent 输出质量标准

| 质量维度 | 标准 |
|---------|------|
| 内容分类 | 100% 准确（根据用户指令关键词） |
| 推送文案 | 口语化、≤3行/条、包含行动引导 |
| 归档完整性 | 每条内容有节点、有推送文案、有推送记录 |
| 查询响应 | 3秒内返回检索结果，格式清晰 |
| 错误处理 | 权限/网络/参数错误有明确指引 |
