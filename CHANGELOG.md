# Changelog

All notable changes to this skill project will be documented in this file.

## [2.0.0] — 2026-06-07

### Added
- **模块0：每日自动实时内容采集** — 核心新增能力
  - SaaS行业资讯自动采集：3轮 WebSearch（综合搜索 + 垂直媒体 + 竞品动态）
  - 飞书产品更新自动采集：4个官方渠道（帮助中心 + 开放平台 + 博客 + 公众号）
  - 48小时内新鲜度过滤（行业资讯）/ 7天内（飞书更新）
  - 采集预览输出格式（含来源URL、发布时间）
  - 人工审核把关机制（预览 → 确认 → 归档）
  - 定时采集调度配置（支持 /loop 和 CronCreate）
- 内容来源标注：所有自动采集内容标注来源 URL 和采集时间
- 补充搜索能力：支持用户定向增补搜索
- 更新每日标准工作流（v2.0）：采集 → 确认 → 手动补充 → 推送

### Changed
- 核心原则更新：从「仅手动录入」升级为「自动采集 + 手动录入双模式」
- 高频指令速查新增 3 条自动采集指令
- 测试数据集新增 4 个自动采集测试场景
- 工作流编排从 3 步扩展为 4 步（新增上午自动采集环节）
- 配置模板新增 auto_collect 调度配置

## [1.0.0] — 2026-06-07

### Added
- 模块1：内容归集与分类（SaaS行业资讯 / 飞书新功能 / 培训资料）

- 推送格式规范（≤3行/条、口语化、行动引导语）
- 推送内容匹配矩阵
- 首次初始化设置流程
- 权限与 Scope 文档
- 错误处理指南
- 完整示例会话
- 项目记忆系统（MEMORY.md）
